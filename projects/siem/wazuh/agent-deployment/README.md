# Agent Deployment – Wazuh

This directory contains Wazuh agent deployment write-ups for endpoints in my SOC homelab.

The purpose of this section is to:
- Deploy Wazuh agents to endpoints
- Confirm agent registration and connectivity
- Verify initial log ingestion and telemetry visibility

Current deployments:
- `windows10/` – Wazuh agent deployment to a Windows 10 endpoint
- `dc01/` – Wazuh agent deployment to a Windows Server domain controller
- `ubuntu-desktop/` – Wazuh agent deployment to a Ubuntu Desktop endpoint
