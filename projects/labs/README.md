# Labs – SOC Homelab

This directory contains hands-on lab projects completed in my SOC homelab.

The purpose of this section is to:

- Build practical defensive security skills through real configurations
- Validate that security events are generated and observable
- Document setup steps, troubleshooting, and lessons learned
- Produce clear evidence (screenshots, notes, sanitized configs) for each lab

Each subdirectory represents a focused lab project (example: `ad-foundation`) with its own README and supporting artifacts.
